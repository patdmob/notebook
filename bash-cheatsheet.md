# Bash

## Header
#!/bin/bash

## Shortcuts

 |||
 ------- | ---------------- 
  CTRL+L | clears the screen  
 CTRL+C  | stops current command 
 CTRL+Z  | suspends current command, to resume: fg (foreground) or bg (background )
 CTRL+D  | logs out of current session, similar to exit 
 !!      | repeat previous command
